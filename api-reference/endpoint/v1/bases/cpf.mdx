<Warning>
  Verifique [Aqui](https://api.thugapplications.xyz/rotas) se a rota está disponíveil Gratuitamente ou apenas para assinantes.
</Warning>

<ParamField query="authorization" type="string" placeholder="Token da sua Conta">
  Seu token. Você pode obter uma chave de API [aqui](https://api.thugapplications.xyz/) fazendo o cadastro e indo na aba Meu Perfil da dashboard.
</ParamField>

<ParamField query="cpf" type="string" placeholder="Número do CPF" required>
  O número do CPF que deseja consultar.
</ParamField>

<ParamField query="page" type="string" placeholder="Exibição de Página ao invés de dados em json">
  true = Exibir página || false = Exibir dados em Json
</ParamField>


### Response

<ResponseField name="status" type="string">
  O status da requisição. Se a requisição foi bem sucedida, o valor será `success` com status code `200`. Caso contrário, o valor será `error` com status code `400`.
</ResponseField>
