## Token de acesso

Nem todas as rotas necessitam de um token de acesso, mas algumas sim. Para obter um token de acesso, você deve obtela pessoalmente com o desenvolvedor da API [XDuke](https://t.me/XDukeBR).
```
EXEMPLO: https://api.thugapplications.xyz/api/cpf?cpf=12345678901&token=SEU_TOKEN
```
